## 项目简介

蜻蜓（QingTing） 是一个经过微调的面向重大慢病的健康管理大模型，它基于Baichuan2-7B 架构，通过在医疗和健康领域的数据集上进行微调，使其在主动健康和支持方面具备更高的性能和专业性。
